Authored by Andrew Ford on August 15, 2022.

## Standards
`afpy` is broken out into **Modules**.
- Advisor
- Configurator
- Exporter

Each **Module** may contain a class file (`module.py`), along with one or more **Submodules**.
- Generator
    - `generator.py`
    - Mermaid
        - `mermaid.py`
    - Markdown
        - `markdown.py`
